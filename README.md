# POWER-BI-Project-on-Executive-Loan-Overview-Dashboard
Executive Loan Overview Dashboard
Project Overview

This Power BI dashboard provides a comprehensive analysis of loan performance, funding trends, borrower profiles, and financial risk indicators. It is designed for executives and stakeholders to monitor lending operations, evaluate borrower behavior, and identify key business insights through interactive visualizations.

**The dashboard delivers a high-level summary of:**

Total loan funding
Average interest rate
Loan status distribution
Borrower home ownership patterns
State-wise loan distribution
Debt-to-Income (DTI) analysis
Verification status insights

Dashboard Features

**1. Loan Status Analysis-**

**Visual representation of loan amounts categorized by:**

Fully Paid
Charged Off
Current

**Insight:**

Most loan amounts belong to fully paid customers, indicating strong repayment behavior, while current loans represent the lowest portion.

**2. Funding Trend Over Time-**

A time-series visualization showing total funding issued across different months in 2021.

**Insight:**

Loan funding experienced significant growth toward the end of the year, with a sharp peak during the final quarter.

**3. Home Ownership Distribution-**

**Donut chart representing borrower home ownership categories:**

Rent
Mortgage
Own
Other
None

**Insight:**

The majority of borrowers belong to the RENT and MORTGAGE categories.

**4. State-wise Loan Amount-**

Horizontal bar chart displaying total loan amount by state.

**Top States by Loan Amount:**

California (CA)
New York (NY)
Texas (TX)
Florida (FL)

**Insight:**

California contributes the highest loan amount among all states.

**5. DTI (Debt-to-Income) Ratio Analysis-**

**Comparison between:**

High DTI
Healthy DTI

**Insight:**

High DTI borrowers are significantly higher, indicating increased financial risk exposure.

**6. Verification Status Analysis-**

Loan amount distribution based on verification categories:

Verified
Source Verified
Not Verified

**Insight:**

Most loans belong to verified customers, improving lending reliability and reducing fraud risk.

**Interactive Filters-**

The dashboard includes slicers for:

Loan Grade (A–G)
Loan Status

These filters allow users to perform dynamic analysis across different borrower categories and risk levels.
